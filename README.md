# ☕ Chai & Chaos — Hostel Food Fight

**Chai & Chaos** is a fast-paced, arcade-style web game built with React. Set in a chaotic college hostel canteen where "Canteen Aunty" has gone missing, players must step up to serve hungry, screaming students before time runs out.

![React](https://img.shields.io/badge/React-20232A?style=for-the-badge&logo=react&logoColor=61DAFB)
![SVG](https://img.shields.io/badge/SVG-FFB13B?style=for-the-badge&logo=svg&logoColor=white)
![Kids-Friendly](https://img.shields.io/badge/Kids--Friendly-Bright%20%26%20Bouncy-green?style=for-the-badge)

## 🎨 Custom SVG Avatars
The game features four unique characters, each hand-drawn in SVG to match specific "roomie" personas:
- **Shubhi:** Wavy highlighted hair & sparkle power 💅
- **Aish:** Traditional red kurta & patient charmer 🗣️
- **Appu:** Iconic cat-eye shades & foodie vibes 🕶️
- **Proxy:** Spiky hair, shades, and "hacker" automation 🤖

## 🎮 Gameplay Features
* **3 Challenging Levels:**
    1.  **Monday Morning:** A sleepy, easy start.
    2.  **Maggi Rush Hour:** High-speed noodle madness.
    3.  **Power Cut Madness:** Candle-lit intensity with faster spawns.
* **Unique Power System:** Each character has a specific ability (e.g., Shubhi earns 2× coins, while Proxy auto-serves every 5th order).
* **Silly Combo Tiers:** Scale from "NICE" to "CHAI CHAMPION" to boost your score.
* **Sticker Book:** A persistent achievement system with 8 unlockable stickers like *Monday Master* and *Blackout Hero*.
* **Floating Power-ups:** Grab ❄️ Freeze, 💰 Money Rain, or 🎉 Party Mode to stay alive.

## 🛠️ Technical Details
- **Responsive Canvas:** Designed with big tap targets and a "mobile-first" feel for kid-friendly interaction.
- **Dynamic CSS Animations:** Uses custom `@keyframes` for "bouncy" cartoon effects, "wiggle" title logos, and "float-up" score indicators.
- **State Management:** Built entirely with React Hooks (`useState`, `useEffect`, `useRef`) for real-time customer spawning and game-loop logic.
- **Zero Assets:** All graphics, including characters and food icons, are rendered via inline SVGs or Emojis, making the game extremely lightweight.

## 🚀 How to Run
1.  Ensure you have [Node.js](https://nodejs.org/) installed.
2.  Create a new React project: `npx create-react-app chai-chaos`.
3.  Replace `App.js` with the `chai-and-chaos.jsx` code.
4.  Run `npm start`.
